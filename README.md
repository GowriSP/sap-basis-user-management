# sap-basis-user-management

## Overview
This repository contains SAP BASIS User Administration activities performed in SAP S/4HANA environments.

## Topics Covered

### User Administration
- SU01 User Creation
- User Modification
- User Lock & Unlock
- Password Reset

### Role Administration
- PFCG Role Assignment
- Role Removal
- Authorization Concepts

### Mass User Maintenance
- SU10 Operations
- Bulk Lock/Unlock
- Bulk Password Reset

### Security
- User Types
- Login Parameters
- Password Policies

## SAP Environment

- SAP S/4HANA 2023
- SAP HANA Database
